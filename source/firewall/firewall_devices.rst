**SIP ALG**
^^^^^^^^^^^^^^^
Most of the time this setting is set to off or disabled and varies.  Rarely this should be enabled. Below is a list a devices that need setting changes to address SIP ALG issues.  

* `ASUS RT-AC66U`_
* `Cisco EA6500`_
* `Sonicwall_TZ_Soho`_
* `Zyxel`_
